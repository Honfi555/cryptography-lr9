# cryptography-lr9
This repository contains an educational implementation of the Diffie-Hellman cipher, on Python3.12

## Генерация простых чисел
Генерация надёжных простых чисел является очень важной частью транспортного протокола Диффи-Хеллмана, поэтому в работе реализована такая генерация.
Объяснение этой генерации можно найти в статье:
[Генерация простых чисел.](https://habr.com/ru/articles/470159)

## Протокол Диффи-Хеллмана
Базовая структура протокола была взята со статьи:
[Алгоритм Diffie-Hellman: Ключ к безопасному общению](https://habr.com/ru/articles/726324)
